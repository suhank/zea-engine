backPropagateValue#

▸ backPropagateValue(value): any

When the value on a Parameter is modified by a user by calling 'setValue, then if any operators are bound, the value of the Parameter cannot be modified directly as it is the result of a computation. Instead, the Parameter calls 'backPropagateValue' on the Operator to cause the Operator to handle propagating the value to one or more of its inputs. to its inputs.

once#

▸ once(eventName, listener): number

Similar to the on method with the difference that when the event is triggered, it is automatically unregistered meaning that the event listener will be triggered at most one time.

Useful for events that we expect to trigger one time, such as when assets load.

const asset = new Asset();asset.once('loaded', () => {  console.log("Yay! the asset is loaded")})
