Class: MultiChoiceParameter

Represents a specific type of parameter, that stores multiple choice(array) values.

i.e.:

const multiChoiceParameter =  new MultiChoiceParameter('InitialXfoMode', GROUP_INITIAL_XFO_MODES.average, [                                 'manual',                                 'first',                                 'average',                                 'global',                               ])//'myParameterOwnerItem' is an instance of a 'ParameterOwner' class.// Remember that only 'ParameterOwner' and classes that extend from it can host 'Parameter' objects.myParameterOwnerItem.addParameter(multiChoiceParameter)

once#

▸ once(eventName, listener): number

Similar to the on method with the difference that when the event is triggered, it is automatically unregistered meaning that the event listener will be triggered at most one time.

Useful for events that we expect to trigger one time, such as when assets load.

const asset = new Asset();asset.once('loaded', () => {  console.log("Yay! the asset is loaded")})
