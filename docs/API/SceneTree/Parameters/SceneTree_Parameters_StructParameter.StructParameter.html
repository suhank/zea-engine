Class: StructParameter

Represents a specific type of parameter, that stores multiple parameters in object format.

i.e.:

const structParam = new StructParameter('MyStructParam')//'myParameterOwnerItem' is an instance of a 'ParameterOwner' class.// Remember that only 'ParameterOwner' and classes that extend from it can host 'Parameter' objects.myParameterOwnerItem.addParameter(structParam)

Events

  • valueChanged: Triggered whenever parameter's value changes.

once#

▸ once(eventName, listener): number

Similar to the on method with the difference that when the event is triggered, it is automatically unregistered meaning that the event listener will be triggered at most one time.

Useful for events that we expect to trigger one time, such as when assets load.

const asset = new Asset();asset.once('loaded', () => {  console.log("Yay! the asset is loaded")})
