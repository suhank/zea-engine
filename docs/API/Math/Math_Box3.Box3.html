Class: Box3

Class representing a box in 3D space. Represents a box in 3D space defined by two Vec3 values which define opposing corners of the box.

Constructors#

constructor#

• new Box3(p0?, p1?)

Creates a Box3 object using Vec3s. In case the parameters are not passed by, their values are pre-defined:

p0 is a Vec2 with Number.POSITIVE_INFINITY

p1 is a Vec2 with Number.NEGATIVE_INFINITY
