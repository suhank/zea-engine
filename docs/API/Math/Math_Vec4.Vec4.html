Class: Vec4

Represents a four-dimensional coordinate. Math types internally store values in Float32Array and expose getters and setters for the component values.

Constructors#

constructor#

• new Vec4(x?, y?, z?, t?)

Creates a Vec4.

The type of values of the (x, y, z, t) coordinates can be Float32Array, Uint32Array, Int32Array and ArrayBuffer.

You can also pass one JSON object parameter.
